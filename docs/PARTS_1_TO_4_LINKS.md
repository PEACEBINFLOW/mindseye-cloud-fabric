# Parts 1–4 Integration

This repo is the **cloud fabric** that ties all prior work together.

- **Part 1** — LAW-T core / time-native language
- **Part 2** — LAW-N network layer
- **Part 3** — Moving Library / Binary Engine / Ledgers / Analytics / Devlog
- **Part 4** — MindsEye SQL epicenter + cloud fabric (this repo)

Data flows:

- Chrome / Android / Agents → LAW-N → Splitter → **MindsEye SQL**
- Binary Engine → **SQL** (patterns table)
- Moving Library → **SQL** (code patterns, agent tables)
- Ledgers / Analytics / Devlog / Dashboards → **read/write via SQL fabric**
